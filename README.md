# KeenKeeper

KeenKeeper is a friendship tracker web application that helps users keep meaningful relationships active. Users can monitor friends, check contact goals, log interactions, browse timeline history, and view friendship analytics.

## Live Features

- View all friends in a responsive card layout
- Open a friend details page with profile, tags, bio, and contact goal
- Log Call, Text, and Video interactions with toast notifications
- Track all interaction history in the Timeline page
- Filter timeline entries by interaction type
- View friendship analytics with a Recharts pie chart
- Custom 404 page for invalid routes
- Responsive design for mobile, tablet, and desktop

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- Recharts
- React Hot Toast
- React Icons
- Vite

## Key Features

1. Dynamic friend cards loaded from a JSON data source
2. Quick Check-In system that adds timeline entries instantly
3. Friendship Analytics chart based on interaction history

## Pages

- Home
- Friend Details
- Timeline
- Stats
- 404 Not Found

## Data Source

Friend data is stored in a local JSON file inside the `public` folder.
